      <h1>Lesson 6: CSS Dimensions & Pseudo-Classes</h1>
      <h2>CSS Dimensions</h2>
      <p>CSS can be used to control dimensions such as visibility, width, and height of HTML elements.</p>
      <ul>
        <li><strong>visibility:</strong> visible | hidden | collapse</li>
        <li><strong>width:</strong> width of an element</li>
        <li><strong>height:</strong> height of an element</li>
        <li><strong>line-height:</strong> controls spacing between lines</li>
        <li><strong>max/min height & width:</strong> set element limits</li>
      </ul>

      <h2>Pseudo-Classes and Links</h2>
      <p>Pseudo-classes are predefined selectors that define a special state of an element.</p>
      <ul>
        <li><code>:link</code> — unvisited link</li>
        <li><code>:visited</code> — visited link</li>
        <li><code>:hover</code> — when hovered</li>
        <li><code>:active</code> — when clicked</li>
        <li><code>:focus</code> — when focused</li>
      </ul>

      <p><strong>Example:</strong></p>
      <code>
        a:link { color: blue; }<br>
        a:visited { color: purple; }<br>
        a:hover { color: green; }<br>
        a:active { color: red; }<br>
        a:focus { background-color: yellow; }
      </code>

      <p><strong>Pseudo-Classes Beyond Links</strong></p>
      <ul>
        <li>They can be used on other HTML elements such as images, paragraphs, and divs.</li>
      </ul>
      <code>
        p:hover { background-color: lightblue; }<br>
        img:hover { border: 3px solid blue; }
      </code>
