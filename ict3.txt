      <h1>Lesson 3: Display Property</h1>
      <p>The CSS display property allows you to control how an element is displayed.</p>

      <p><strong>Block Level Elements</strong><br>
      Block level elements take up as much space as possible by default. They start on a new line and stack vertically. Examples include:</p>
      <ul>
        <li>&lt;p&gt;</li>
        <li>&lt;ol&gt;, &lt;ul&gt;, &lt;dl&gt;</li>
        <li>All heading tags</li>
        <li>&lt;article&gt;, &lt;section&gt;, &lt;div&gt;</li>
      </ul>

      <p><strong>Inline Level Elements</strong><br>
      Inline elements display in a line without forcing the next text to a new line. Examples include:</p>
      <ul>
        <li>&lt;a&gt;</li>
        <li>&lt;strong&gt;, &lt;em&gt;, &lt;b&gt;, &lt;i&gt;, &lt;q&gt;, &lt;mark&gt;</li>
        <li>&lt;span&gt;</li>
      </ul>
      <p><em>Note: Inline elements cannot contain block-level elements.</em></p>

      <p><strong>Div Tag</strong><br>
      Div divides the contents of a web page into individual sections. It allows you to apply styles to different sections of your webpage. It is a block-level element and usually applied to block HTML content.</p>

      <p><strong>Span Tag</strong><br>
      Span is used to emphasize words or sentences within a paragraph. It is an inline element used for smaller parts of content.</p>

      <p><strong>Span and Div Tags</strong><br>
      They have no default styling. Used with CSS to format parts of the content.
      &lt;div&gt; has line breaks before and after; &lt;span&gt; does not.</p>
