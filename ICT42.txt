      <h1>Lesson 2: Where to Place Scripts</h1>

      <p><strong>JavaScript scripts</strong> can be placed in different parts of an HTML document depending on how the developer wants the script to run.</p>

      <p><strong>1. Inside the Head Section</strong></p>
      <ul>
        <li>Scripts can be placed inside the &lt;head&gt; tag.</li>
        <li>Usually used for functions that will be called later in the page.</li>
        <li>The script runs before the webpage content loads.</li>
      </ul>

      <p><strong>2. Inside the Body Section</strong></p>
      <ul>
        <li>Scripts can also be placed inside the &lt;body&gt; tag.</li>
        <li>This is often placed at the bottom of the body.</li>
        <li>It allows the HTML content to load first before the script runs.</li>
      </ul>

      <p><strong>3. External JavaScript File</strong></p>
      <ul>
        <li>JavaScript code can be written in a separate .js file.</li>
        <li>The file is connected to HTML using the &lt;script src="file.js"&gt; tag.</li>
        <li>This method helps keep code organized and easier to maintain.</li>
      </ul>

      <p><strong>Advantages of Using External Scripts</strong></p>
      <ul>
        <li>Makes code cleaner and more organized</li>
        <li>Allows the same script to be used on multiple pages</li>
        <li>Improves website maintenance and updates</li>
      </ul>
