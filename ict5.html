Lesson 5: CSS Selectors

CSS selectors are used to “find” or “select” the HTML elements that you want to style.

They are divided into six main types:

  • Class selector
  • Element (Tag) selector
  • ID selector
  • Universal selector
  • Group selector
  • Attribute selector

Element Selector

Selects specific HTML elements such as <p>, <div>, or <h1> and applies CSS to them.

p { color: blue; }

ID Selector

Used to apply a style to one unique element on a page. Defined by #.

#title { font-size: 30px; color: red; }
<h1 id="title">Welcome</h1>

Universal Selector

* { margin: 0; padding: 0; box-sizing: border-box; }

Group Selector

h1, h2, h3 { color: blue; }

Attribute Selector

input[type="text"] { background-color: yellow; }

When to Use Each Selector

  • Class Selector (.) — reusable for multiple elements
  • ID Selector (#) — unique element only
  • Element Selector — targets all of the same element
  • Universal Selector (*) — applies to all elements
  • Group Selector — saves time and code
  • Attribute Selector — targets specific attributes
