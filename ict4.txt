      <h1>Lesson 4: CSS Rule & Classes</h1>
      <p>A <strong>CSS rule</strong> has two main parts: a selector and one or more style declarations.</p>

      <p><strong>Basic format of a CSS style rule:</strong></p>
      <p>For Inline Style Rule:</p>
      <code>&lt;tag style="style declaration"&gt;<br>
      &lt;tag style="prop1:value; prop2:value; prop3:value;"&gt;</code>

      <p>For Document-level and External Style Sheets:</p>
      <code>selector { style declarations }<br>
      selector { prop1:value; prop2:value; prop3:value; }</code>

      <h2>Style Properties</h2>
      <p>Properties in CSS are grouped into six categories:</p>
      <ul>
        <li>Fonts</li>
        <li>Colors and Backgrounds</li>
        <li>Text</li>
        <li>Boxes and Layout</li>
        <li>Lists</li>
        <li>Tag Classifications</li>
      </ul>

      <h2>Classes</h2>
      <p><strong>What are Classes?</strong></p>
      <ul>
        <li>A class is a user-defined selector used to apply style rules to specific HTML tags.</li>
        <li>They help control individual element formatting.</li>
        <li>They are reusable and flexible, allowing multiple elements to share the same style.</li>
        <li>Classes are defined by a dot (.) before the class name.</li>
      </ul>

      <h2>How to Define and Use Classes</h2>
      <p>Two ways to write class selectors:</p>
      <ul>
        <li><code>p.golden { color: blue; }</code> → Element + class selector</li>
        <li><code>.sodapop { color: red; }</code> → Class selector only</li>
      </ul>
      <p>Apply a class to an element:</p>
      <code>&lt;p class="golden"&gt;This text is blue.&lt;/p&gt;</code>

      <p><strong>Note:</strong> One element can have multiple classes, and one class can be applied to many elements.</p>

      <h2>Class Selector</h2>
      <ul>
        <li>Class selector (.classname): Reusable across many elements.</li>
        <li>Element + Class selector (p.classname): Gives precise control over styling a specific type of element.</li>
      </ul>
